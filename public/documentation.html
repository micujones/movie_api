Get genre information by name /movies/genre/:Genre GET None

Returns a description of the user-provided genre. If unsuccessful, returns a 500 error stating, "No movies match the [user-provided] genre."

Success example.

"Action films are fast-paced movies characterized by intense sequences of physical feats, chases, battles, and stunts, often revolving around high-stakes conflicts and heroic protagonists."
Get director information by name /movies/director/:Director GET None

Returns a JSON object with the director's name and bio.

{
  "name": "Christopher Nolan",
  "bio": "Christopher Nolan is a critically acclaimed filmmaker known for his thought-provoking and visually stunning films, often exploring complex themes of identity and reality."
}

Create new user /users POST

JSON object with new user's data (required fields). 

{
  "username": "FlorencePizza",
  "password": "FourFireHazards444",
  "email": "florentine@example.com",
  "birthday": "1983-04-24",
  "favoriteMovies": []
}

New user is created, and then a 200 status and a JSON object with the user's information (e.g., generated ID) is returned.

Success example:

{
  "username": "FlorencePizza",
  "password": "FourFireHazards444",
  "email": "florentine@example.com",
  "birthday": "1983-04-24T00:00:00.000Z",
  "favoriteMovies": [],
  "_id": "676f01fbf9acd94021a88be4",
  "__v": 0
}
User updates their information /users/:Username PUT

JSON object with updated user's data.

{
  "username": "FlorencePizza",
  "password": "aStrongerPassword555",
  "email": "florentine@example.com",
  "birthday": "1983-04-24"
}

In this example, the user's password is updated and a JSON object with the user's updated information is returned.

Success example:

{
  "_id": "676f01fbf9acd94021a88be4",
  "username": "FlorencePizza",
  "password": "aStrongerPassword555",
  "email": "florentine@example.com",
  "birthday": "1983-04-24T00:00:00.000Z",
  "favoriteMovies": [],
  "__v": 0
}
Get list of user's favorite movies /users/:Username/movies GET None

Returns a JSON array of the titles and corresponding IDs of the user's favorite movies. If "favoriteMovies" is empty, returns a 400 error stating "@[Username] has not added any movies." If another error occurs, returns a 500 error stating.

Success example (/users/wise_old_wizard/movies):

[ "676d807a2c4cb1f883f66563" ]

Add movie to user's favorites /users/:Username/movies/:MovieID POST None

Returns a JSON object of the user's information, including the added movie's ID in the favoriteMovies array. If the movie cannot be added or the user does not exist, returns a 500 error.

Success example (/users/wise_old_wizard/movies/676d807a2c4cb1f883f66567): 

{
  "_id": "676d80bd2c4cb1f883f6656e",
  "name": "Gandalf the Grey",
  "username": "wise_old_wizard",
  "email": "gandalf@middle-earth.net",
  "password": "staffofpower123",
  "birthday": "1000-01-01T00:00:00.000Z",
  "favoriteMovies": [
      "676d807a2c4cb1f883f66563",
      "676d807a2c4cb1f883f66567"
  ]
}
Delete user by username /users/:Username DELETE None

Returns a statement confirming the user with the provided username has been deleted. If the user does not exist, returns a 400 error stating, "[Username] was not found."

Success example (/users/ToBeDeleted):

ToBeDeleted was deleted.
Delete movies from favorites /users/:Username/movies/:MovieID DELETE None

Returns a JSON object of the user's information, confirming the movie is removed from the favoriteMovies array. If the movie cannot be added or the user does not exist, returns a 500 error.

Success example: (users/wise_old_wizard/movies/676d807a2c4cb1f883f66567) 

{
  "_id": "676d80bd2c4cb1f883f6656e",
  "name": "Gandalf the Grey",

  "email": "gandalf@middle-earth.net",
  "password": "staffofpower123",
  "birthday": "1000-01-01T00:00:00.000Z",
  "favoriteMovies": [
      "676d807a2c4cb1f883f66563"
  ]
}
